# Sistema de Gestion de Estacionamiento

Proyecto universitario en Java (Swing).

## Ejecutar

1. Compilar:
   - javac -d bin -sourcepath src src/com/estacionamiento/gui/ParkingApp.java
2. Ejecutar:
   - java -cp bin com.estacionamiento.gui.ParkingApp

## Release

El ejecutable se entrega en la carpeta /release como un archivo .jar.
